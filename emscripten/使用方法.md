# 生成

准备开发环境，执行`./script/build-emscripten.bat`即可生成如下文件

```
out/emscripten/Release/
  index.html
  index.js
  index.wasm
  index.data
```

# 运行服务器

准备python环境，在out/emscripten/Release/使用wasm_server.py启动服务器，将资源拷贝到out/emscripten/Release/下

# 打开浏览器运行

浏览器输入`https://localhost:5500/index.html?game=data.xp3`即可开始游戏
